Introducción

¿Que es Git?

Git es una herramienta libre y de codigo abierto diseñada para el control y manejo de versiones de codigo, permite organizar desde los más pequeños hasta los más grandes proyectos, de una forma rapida y eficaz.

Página Web de Git: https://git-scm.com/

¿Que es GitHub?

GitHub es una plataforma de desarrollo inspirada en la forma de organizar tu trabajo. Básada en el sistema de versiones de Git, en esta plataforma podras alojar, revisar, compartir y counstruir tu proyecto con millones de usuarios en la nube.

Página Web de GitHub: https://github.com/
